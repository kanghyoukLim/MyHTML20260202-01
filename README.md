# 오늘 뭐 먹지? (Menu Recommender)

간단한 메뉴 추천 웹 애플리케이션입니다.

## 기능
- **카테고리 선택**: 한식, 중식, 양식, 분식 중 하나를 선택할 수 있습니다.
- **메뉴 추천**: 선택한 카테고리에서 랜덤으로 메뉴를 추천해줍니다.
- **레시피 제공**: 추천된 메뉴의 재료와 조리 방법을 알려줍니다.

## 실행 방법
이 프로젝트는 정적 웹사이트입니다. `index.html`을 웹 브라우저로 열어서 확인하거나, 로컬 서버를 통해 실행할 수 있습니다.

## 기술 스택
- HTML5
- CSS3
- Vanilla JavaScript
